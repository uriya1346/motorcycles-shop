Motorcycle's API Documentation

  • Get 9 products by page number: /motorcycle/?page=number(GET)
  • Get X product's per one page: (default:9) /motorcycle/?perpage=X(GET)
  • Search products by name or information: /motorcycle/search/?s=name(GET)
  • Get products by Category-name: /motorcycle/cat/:category(GET)
  • Price range: /motorcycle/prices/?max=x&min=y&page=z(GET)
  • Sort by... (default:"_id"): /motorcycle/?sort=name(GET)
  • Sort from smaller to bigger (default:bigger-smaller): /motorcycle/?r=yes(GET)

Sign Up: /users(POST)

  • User(required)
  • Email(required)
  • Password(required)
  • Role

Sign In: /users/login(POST)

  • Email(required)
  • Password(required)

Information -(only for register user with token)

User Info: /users/userInfo(GET)

Add Product-(only for register user with token) /motorcycle(POST)

  • Name(required)
  • Price(required)
  • Info(required)
  • Cat(required)
  • Img (URL)

Edit Product-(only for register user with token) /motorcycle/:idEdit(PUT)

  • Name(required)
  • Price(required)
  • Info(required)
  • Cat(required)
  • Img (URL)

Delete Product-(only for register user with token) /motorcycle/:idDel(POST)
